PubMed Popup Demo

This demonstrates the new PubMed popup functionality. Hover over the PMID link below to see the popup with article information.

CKD in Diabetes Guidelines

PMID: 40382193

What happens when you hover:
  • A popup appears with article details
  • Title, authors, journal, and publication year
  • DOI link and direct PubMed link
  • Abstract (when available)
  • Results are cached for 24 hours

Features:

  • API Integration: Uses NCBI E-utilities esummary API
  • Caching: 24-hour cache prevents repeated API calls
  • Smart Positioning: Popup positions itself to stay on screen
  • Mobile Responsive: Adapts to mobile screens
  • Error Handling: Graceful fallback if API fails
  • Performance: Lazy loading and efficient DOM handling

Note: This is a demo page. The actual functionality is integrated into the main chat application.
