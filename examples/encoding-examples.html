üìù Code Examples

Method 1: Using encodeURIComponent() (Recommended)

// Encode a single URL const backURL = 'https://brightbean.io/page?id=123&ref=test'; const encoded = encodeURIComponent(backURL); console.log(encoded); // Output: https%3A%2F%2Fbrightbean.io%2Fpage%3Fid%3D123%26ref%3Dtest // Build the full chat URL const chatURL = `?doc=smh&back-button=${encodeURIComponent(backURL)}`; console.log(chatURL); // Output: ?doc=smh&back-button=https%3A%2F%2Fbrightbean.io%2Fpage%3Fid%3D123%26ref%3Dtest

Method 2: Using URLSearchParams (Automatic Encoding)

// URLSearchParams automatically encodes values const params = new URLSearchParams(); params.set('doc', 'smh'); params.set('back-button', 'https://brightbean.io/page?id=123&ref=test'); const chatURL = `?${params.toString()}`; console.log(chatURL); // Output: ?doc=smh&back-button=https%3A%2F%2Fbrightbean.io%2Fpage%3Fid%3D123%26ref%3Dtest

Method 3: Helper Function (Best Practice)

// Create a reusable function function buildChatURL(docSlug, backURL) { const params = new URLSearchParams(); params.set('doc', docSlug); if (backURL) { params.set('back-button', backURL); } return `?${params.toString()}`; } // Usage const url1 = buildChatURL('smh', 'https://brightbean.io'); const url2 = buildChatURL('uhn', 'https://brightbean.io/page?id=123'); const url3 = buildChatURL('smh', null); // No back button

üåç Real-World Use Cases

1. Dynamic Back Link from Current Page

// Use current page as back link const currentURL = window.location.href; const chatURL = `/chat?doc=smh&back-button=${encodeURIComponent(currentURL)}`; // Or use referrer const referrer = document.referrer || 'https://brightbean.io'; const chatURL = `/chat?doc=smh&back-button=${encodeURIComponent(referrer)}`;

2. Embedding in HTML

<!-- Simple URL (no encoding needed) --> <iframe src="/chat?doc=smh&back-button=https://brightbean.io"></iframe> <!-- Complex URL (must be encoded) --> <iframe src="/chat?doc=smh&back-button=https%3A%2F%2Fbrightbean.io%2Fpage%3Fid%3D123"></iframe> <!-- Generate dynamically with JavaScript --> <script> const backURL = 'https://brightbean.io/page?id=123'; const iframe = document.createElement('iframe'); iframe.src = `/chat?doc=smh&back-button=${encodeURIComponent(backURL)}`; document.body.appendChild(iframe); </script>

3. Server-Side (Node.js/Express)

// Express route that generates chat URLs app.get('/embed-chat', (req, res) => { const backURL = req.query.return_url || 'https://brightbean.io'; const docSlug = req.query.doc || 'smh'; // Build URL with proper encoding const params = new URLSearchParams(); params.set('doc', docSlug); params.set('back-button', backURL); const chatURL = `/chat?${params.toString()}`; res.redirect(chatURL); });
